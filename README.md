# Personal Page

This web page is designed and built as a portfolio to showcase my personal projects. It serves as a central hub to display my work, skills, and contact information.

## Built With

The website is built using the following technologies:

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool for modern web development.
- **Contentful**: A headless CMS for managing content.
- **Formspree**: A service for handling form submissions.

### Dependencies

The project relies on the following key dependencies:

- `@emotion/react`: For CSS-in-JS styling.
- `@emotion/styled`: For styled components.
- `@formspree/react`: For integrating Formspree forms.
- `@mui/material`: For Material-UI components.
- `autoprefixer`: For adding vendor prefixes to CSS.
- `contentful`: For fetching content from Contentful.
- `postcss`: For transforming CSS with JavaScript.
- `react`: The core React library.
- `react-dom`: For rendering React components in the DOM.
- `react-icons`: For using popular icons in the project.
- `react-router-dom`: For client-side routing.
- `react-scroll`: For smooth scrolling functionality.
- `react-spinners`: For loading spinners.
- `tailwindcss`: For utility-first CSS styling.

## Features

- **Responsive Design**: The website is fully responsive and works seamlessly on all devices.
- **Dynamic Content**: Content is managed through Contentful, making it easy to update.
- **Contact Form**: Integrated with Formspree for handling form submissions.
- **Smooth Scrolling**: Enhanced user experience with smooth scrolling navigation.
- **Custom Styling**: Utilizes Tailwind CSS and Emotion for a modern and clean design.
